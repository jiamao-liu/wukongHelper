## 软件简介
- 软件初衷：本软件期望设计成一款游戏数据监控、内嵌游戏攻略软件。
- 市面上已经有很多黑神话悟空的修改器，但是我个人还是不太喜欢使用修改器来快速通关游戏，但是本软件已经实现了游戏数据的读取
- 如果你想基于本项目制作一款修改器，应该也是很容易的
